📌 빠른 명령 실행
1. ! 프리픽스로 터미널 바로 실행
!git status처럼 입력하면 bash가 즉시 실행되어 대기 시간과 토큰 소모 감소
7. -p 옵션으로 Headless 모드
대화형 화면 없이 명령어처럼 실행, CI/CD 파이프라인에 통합 가능
15. YOLO 모드
claude --dangerously-skip-permissions로 권한 확인 없이 바로 실행 (주의 필요)
27. sandbox 모드
실행 범위를 미리 정해두고 그 안에서는 권한 확인 없이 작업, YOLO보다 안전
🔄 세션 관리
4. 세션 이어가기
claude --continue: 직전 대화 이어가기
claude --resume: 지난 세션 목록에서 선택
9. 원격 세션
&를 앞에 붙이면 웹으로 전송
claude --teleport 세션ID로 로컬에서 불러오기
11. 세션 이름 관리
/rename api-migration으로 이름 붙이기
/resume api-migration으로 특정 세션 불러오기
⚡ 단축키 & 편집
2. Esc 두 번으로 되돌리기
대화, 코드, 또는 둘 다 이전 상태로 복원
6. /vim으로 프롬프트 편집
vim 방식으로 키보드만으로 프롬프트 수정
13. Ctrl+S로 프롬프트 임시 저장
입력 중인 프롬프트 저장 및 자동 복원
22. Ctrl+R로 프롬프트 검색
과거 입력한 프롬프트 역방향 검색 및 재사용
29. Plan mode (Shift+Tab 두 번)
코드 수정 없이 읽기, 검색, 분석만 진행
💾 컨텍스트 & 메모리
5. # 프리픽스로 메모리 저장
# Always use bun instead of npm으로 선호사항 저장
8. /context로 토큰 사용 확인
시스템 프롬프트, MCP 서버, 메모리 파일 등 컨텍스트 사용 현황 확인
17. @ 멘션으로 컨텍스트 추가
파일, 디렉터리, MCP 서버를 빠르게 대화에 추가
23. /init으로 온보딩 문서 자동 생성
코드베이스 분석 후 빌드 방법, 테스트 명령 등을 CLAUDE.md로 정리
🤖 고급 AI 기능
3. Extended Thinking 활성화
API에 thinking: { type: "enabled", budget_tokens: 5000 } 추가
12. 키워드로 사고 깊이 조절
"think", "think hard", "ultrathink" 등으로 사고 토큰 양 조절
25. Subagents로 병렬 처리
각 200k 컨텍스트를 가진 보조 에이전트로 작업 분산
🛠️ 자동화 & 커스터마이징
16. Hooks로 실행 흐름 제어
특정 시점에 자동 실행되는 쉘 명령 설정
20. Agent Skills
특정 작업 방법을 폴더 단위로 묶어 재사용
21. 플러그인
/plugin install my-setup으로 작업 환경 한 번에 설치
26. 프롬프트를 명령어로 저장
마크다운 파일로 저장 후 /explain 같은 슬래시 명령으로 실행
📊 모니터링 & 관리
10. /stats로 사용 기록 확인
작업 횟수, 사용 패턴 등 통계 확인
14. /statusline 커스터마이징
Git 상태, 모델, 토큰량 등 필요한 정보만 하단 바에 표시
28. /usage로 한도 확인
토큰 사용량 확인 및 /extra-usage로 제한 해제
🌐 브라우저 & IDE 통합
19. /chrome으로 브라우저 제어
페이지 이동, 클릭, 콘솔 에러 확인, DOM 검사 등 프런트엔드 디버깅
30. LSP 연결
오류/경고 확인, 정의 이동, 참조 찾기 등 IDE 기능 사용
📤 내보내기 & 제안
18. 자동 프롬프트 제안
작업 완료 후 다음 작업 제안, Tab/Enter로 실행
24. /export로 대화 기록 문서화
프롬프트, 응답, 도구 호출을 마크다운으로 내보내기
🚀 개발자용
31. Claude Agent SDK
Claude Code의 에이전트 루프, 도구 사용, 컨텍스트 관리를 SDK로 제공
서비스나 자동화에 통합 가능한 커스텀 에이전트 제작
