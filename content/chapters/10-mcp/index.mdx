## MCP란?

MCP(Model Context Protocol)는 Claude가 외부 도구와 데이터 소스에 접근할 수 있게 해주는 프로토콜입니다. MCP 서버를 설정하면 Claude가 Notion 문서를 읽거나, Slack 메시지를 보내거나, GitHub 이슈를 관리하는 등의 작업을 수행할 수 있습니다.

<Callout type="info">
Boris Cherny는 Claude에게 사용할 툴 권한을 모두 제공합니다. Slack 검색, BigQuery 쿼리 실행, Sentry 에러 로그 확인 등 다양한 도구를 MCP로 연결합니다.
</Callout>

## MCP 서버 설정

```json title=".claude/settings.json"
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "your-token-here"
      }
    },
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/allowed/dir"]
    }
  }
}
```

## 인기 MCP 서버

| 서버 | 용도 |
|------|------|
| `server-github` | GitHub 이슈, PR 관리 |
| `server-slack` | Slack 메시지 검색, 전송 |
| `server-filesystem` | 파일 시스템 접근 |
| `server-postgres` | PostgreSQL 쿼리 |
| `server-notion` | Notion 문서 접근 |

## 실사용 시나리오

### GitHub 이슈 관리
```
GitHub 이슈 #42를 확인하고, 해결 방법을 제안해줘.
코드 수정이 필요하면 바로 수정하고 PR도 생성해줘.
```

### Slack 연동
```
#dev-channel에서 최근 에러 보고를 검색하고,
관련 코드를 찾아서 수정 방안을 알려줘.
```

### 데이터베이스 쿼리
```
PostgreSQL에서 최근 24시간 동안 실패한 주문을 조회하고,
원인을 분석해줘.
```

## `@` 멘션으로 MCP 서버 활성화

대화 중에 `@` 멘션으로 MCP 서버를 즉시 활성화하거나 비활성화할 수 있습니다.

```
@github 이 리포의 열린 이슈 목록을 보여줘
```
