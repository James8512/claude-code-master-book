## IDE와 함께 사용하기

Claude Code는 터미널 기반이지만, IDE와 함께 사용하면 더 강력해집니다.

## VS Code 통합

VS Code 터미널에서 Claude Code를 실행하면 에디터와 자연스럽게 연동됩니다. Claude이 수정한 파일이 에디터에 바로 반영됩니다.

## JetBrains 플러그인

JetBrains IDE(IntelliJ, WebStorm 등)에서도 Claude Code를 사용할 수 있습니다. 내장 터미널에서 실행하거나 전용 플러그인을 설치합니다.

## `/chrome`으로 브라우저 제어

```bash
/chrome
```

Claude Code에서 브라우저를 직접 제어할 수 있습니다.

- 페이지 이동
- 버튼 클릭
- 콘솔 에러 확인
- DOM 검사

```
/chrome으로 localhost:3000을 열고, 로그인 버튼을 클릭한 후
콘솔에 에러가 있는지 확인해줘
```

프론트엔드 디버깅에서 "버그 고치고 실제로 되는지 확인해줘" 같은 요청을 한 번에 처리할 수 있습니다.

## LSP 연결

Claude Code에 LSP(Language Server Protocol)를 연결하면 IDE 수준의 코드 이해력을 갖습니다.

- 오류와 경고 즉시 확인
- 정의 이동 (Go to Definition)
- 참조 찾기 (Find References)
- 타입 정보 확인

타입 정보와 문서까지 바로 보여줘서, 별도 에디터 없이도 코드 구조를 정확히 파악할 수 있습니다.

## 자동 프롬프트 제안

작업이 끝나면 Claude가 다음에 할 만한 요청을 회색 글씨로 제안합니다.

- <KeyCombo keys={["Tab"]} /> — 제안을 불러와 수정
- <KeyCombo keys={["Enter"]} /> — 제안을 바로 실행

흐름을 끊지 않고 연속적으로 작업할 수 있습니다.
