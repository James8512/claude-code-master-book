## Claude Code란?

Claude Code는 Anthropic이 만든 **에이전틱 코딩 도구**입니다. 터미널에서 직접 실행되며, 여러분의 코드베이스를 이해하고, 파일을 편집하고, 명령어를 실행하며, 심지어 Git 작업까지 자동으로 처리합니다.

기존의 코드 자동완성 도구와는 차원이 다릅니다. Claude Code는 **프로젝트 전체 맥락**을 이해하고, 여러 파일에 걸친 변경사항을 한 번에 처리할 수 있는 진정한 AI 코딩 에이전트입니다.

<Callout type="info">
Claude Code는 단순한 코드 자동완성이 아닙니다. 코드를 읽고, 이해하고, 수정하고, 실행하고, 검증까지 하는 **자율형 코딩 에이전트**입니다.
</Callout>

## 핵심 기능

### 코드베이스 전체 이해
프로젝트 구조, 파일 간 의존성, 코드 패턴을 분석하여 맥락에 맞는 수정을 제안합니다.

### 멀티파일 편집
하나의 지시로 여러 파일을 동시에 수정할 수 있습니다. 컴포넌트 리팩토링, API 엔드포인트 추가, 테스트 작성 등을 한 번에 처리합니다.

### 터미널 명령 실행
빌드, 테스트, 린팅 등 개발에 필요한 명령어를 직접 실행하고 결과를 분석합니다.

### Git 워크플로우 자동화
커밋 메시지 작성, PR 생성, 머지 충돌 해결까지 Git 관련 작업을 자동화합니다.

### MCP 서버 연동
Notion, Slack, GitHub Issues 등 외부 도구와 연동하여 작업 범위를 확장합니다.

## 왜 Claude Code를 써야 하는가?

### 생산성 폭발
Boris Cherny(Claude Code 창시자)는 15개 세션을 병렬로 실행하며 주당 PR 100개를 혼자 처리합니다. Claude Code는 단순 코딩 보조가 아닌 **업무 방식 자체를 변화**시킵니다.

### 컨텍스트 유지
`--continue`, `--resume` 옵션으로 이전 대화를 이어갈 수 있어 맥락을 잃지 않습니다.

### 학습 곡선이 낮음
자연어로 지시하면 됩니다. "이 함수를 리팩토링해줘", "이 버그 고쳐줘"처럼 일상적인 한국어로 작업을 요청할 수 있습니다.
