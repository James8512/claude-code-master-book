## 빠른 실행

| 팁 | 설명 |
|-----|------|
| `!명령어` | bash 즉시 실행, 추론 없이 바로 실행 |
| `claude -p "질문"` | Headless 모드, CI/CD 통합에 유용 |
| `claude --sandbox` | 허용 범위 안에서 권한 확인 없이 작업 |
| `claude --dangerously-skip-permissions` | YOLO 모드 (주의 필요) |

## 세션 관리

| 팁 | 설명 |
|-----|------|
| `claude --continue` | 직전 대화 이어가기 |
| `claude --resume` | 세션 목록에서 선택 |
| `/rename 이름` | 세션에 이름 붙이기 |
| `& 프롬프트` | 백그라운드/웹으로 전송 |
| `claude --teleport ID` | 원격 세션을 로컬로 불러오기 |

## 단축키

| 단축키 | 기능 |
|--------|------|
| <KeyCombo keys={["Esc", "Esc"]} /> | 이전 상태로 되돌리기 |
| `/vim` | Vim 방식 프롬프트 편집 |

| <KeyCombo keys={["Ctrl", "R"]} /> | 프롬프트 역방향 검색 |
| <KeyCombo keys={["Shift", "Tab"]} /> × 2 | Plan 모드 토글 |
| `@파일명` | 파일/디렉토리 컨텍스트 추가 |
| <KeyCombo keys={["Tab"]} /> | 자동 제안 수락 |

## 컨텍스트 & 메모리

| 팁 | 설명 |
|-----|------|

| `@파일` | 파일을 컨텍스트에 추가 |
