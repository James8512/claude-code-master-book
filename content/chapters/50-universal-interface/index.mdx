## 컴퓨터의 범용 인터페이스

Claude Code는 코딩 도구를 넘어 컴퓨터와 상호작용하는 범용 인터페이스입니다. 터미널에서 실행할 수 있는 모든 작업을 자연어로 요청할 수 있습니다.

<Callout type="tip" title="ykdojo의 철학">
"컴퓨터에서 하고 싶은 일이 있으면, 일단 Claude Code에게 물어보세요." ykdojo는 Claude Code를 단순한 코딩 도구가 아니라, 컴퓨터를 조작하는 보편적인 인터페이스로 사용합니다.
</Callout>

## 영상 편집 (ffmpeg)

전문 영상 편집 소프트웨어 없이도 ffmpeg 명령어로 다양한 영상 처리가 가능합니다. 복잡한 ffmpeg 옵션을 외울 필요 없이 자연어로 요청하면 됩니다.

```
> 이 영상에서 0:30부터 2:15까지 잘라내고,
  해상도를 1080p로 변환하고,
  자막 파일을 하드코딩해줘
```

```bash
$ ffmpeg -i input.mp4 -ss 00:00:30 -to 00:02:15 \
  -vf "scale=1920:1080,subtitles=subs.srt" \
  -c:a copy output.mp4
```

```
> 이 폴더의 모든 MOV 파일을 MP4로 변환하고,
  파일 크기를 50% 줄여줘
```

## 오디오 전사 (Whisper)

OpenAI의 Whisper 모델을 활용하여 오디오를 텍스트로 변환합니다.

```
> 이 회의 녹음 파일을 텍스트로 전사해줘.
  한국어로 전사하고, 타임스탬프도 포함해줘
```

```bash
$ whisper meeting.mp3 --language ko --model medium \
  --output_format txt --output_dir ./transcripts
```

회의록을 자동으로 작성하거나, 팟캐스트 에피소드의 스크립트를 추출하는 데 활용할 수 있습니다.

## 디스크 정리

```
> 홈 디렉토리에서 100MB 이상인 파일을 찾아서 크기 순으로 정렬해줘.
  숨김 파일도 포함하고, 각 파일의 마지막 접근 시간도 보여줘
```

```
> node_modules 폴더를 모두 찾아서 총 용량을 계산해줘.
  6개월 이상 사용하지 않은 프로젝트의 node_modules는 삭제해도 되는지 확인해줘
```

## 그 외 활용 사례

### 이미지 처리

```
> images 폴더의 모든 PNG 파일을 WebP로 변환하고,
  각각 가로 800px로 리사이즈해줘. 원본은 유지해줘
```

### 네트워크 진단

```
> 인터넷 연결이 느린 것 같아.
  DNS 응답 시간, 주요 서버 ping, 대역폭을 측정해줘
```

### 시스템 모니터링

```
> 현재 시스템의 CPU, 메모리, 디스크 사용량을 확인하고,
  비정상적으로 많은 리소스를 사용하는 프로세스가 있으면 알려줘
```

### PDF 처리

```
> 이 PDF에서 텍스트를 추출해줘.
  표 형식은 CSV로 변환하고, 이미지는 별도 파일로 저장해줘
```

<Callout type="info" title="도구가 없으면 설치까지">
필요한 CLI 도구가 설치되어 있지 않으면, Claude Code가 설치 방법을 안내하거나 직접 설치할 수 있습니다. "ffmpeg가 없으면 설치해줘"라고 추가하면 됩니다.
</Callout>
