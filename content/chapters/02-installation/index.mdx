## 설치하기

Claude Code는 npm으로 설치합니다. Node.js 18 이상이 필요합니다.

```bash title="Claude Code 설치"
npm install -g @anthropic-ai/claude-code
```

설치 후 터미널에서 `claude`를 입력하면 바로 시작됩니다.

```bash
claude
```

첫 실행 시 Anthropic 계정으로 로그인하라는 안내가 나옵니다. 브라우저에서 인증을 완료하면 바로 사용할 수 있습니다.

<Callout type="tip">
Node.js가 없다면 `nvm`으로 설치하는 것을 추천합니다. `nvm install 20 && nvm use 20`으로 빠르게 설정할 수 있습니다.
</Callout>

## 터미널 설정

Claude Code는 터미널 기반 도구이므로 터미널 환경이 중요합니다.

### 추천 터미널 설정

- **macOS**: iTerm2 또는 기본 터미널 + Oh My Zsh
- **Windows**: Windows Terminal + WSL2
- **Linux**: 기본 터미널 + tmux

### 터미널 폰트

코드와 한글이 잘 보이려면 Nerd Font 계열을 추천합니다.

```bash title="JetBrains Mono Nerd Font 설치 (macOS)"
brew install --cask font-jetbrains-mono-nerd-font
```

## 권한 설정

Claude Code는 파일 읽기/쓰기, 명령어 실행 등의 권한이 필요합니다. 매번 묻는 것이 번거롭다면 `/permissions`를 사용합니다.

```bash
/permissions
```

<Callout type="warning">
`--dangerously-skip-permissions` (YOLO 모드)는 모든 권한을 건너뛰므로, 신뢰할 수 있는 환경에서만 사용하세요. 자세한 내용은 Ch.12 보안 챕터에서 다룹니다.
</Callout>

### sandbox 모드

sandbox 모드를 사용하면 허용된 범위 안에서만 Claude가 작업합니다. YOLO 모드만큼 빠르면서도 훨씬 안전합니다.

```bash title="sandbox 모드로 실행"
claude --sandbox
```

## `-p` 옵션으로 Headless 모드

대화형 UI 없이 명령어처럼 실행할 수도 있습니다. CI/CD 파이프라인이나 스크립트에 통합할 때 유용합니다.

```bash title="Headless 모드 예시"
claude -p "이 프로젝트의 README를 한국어로 번역해줘"
```

```bash title="파이프와 함께 사용"
git diff | claude -p "이 변경사항을 요약해줘"
```
