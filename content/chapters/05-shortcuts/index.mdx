## `!` 프리픽스로 터미널 바로 실행

"git status 실행해줘"라고 묻지 마세요. `!`를 붙이면 bash가 즉시 실행됩니다.

```bash
!git status
!pwd
!npm test
```

모델 추론 없이 바로 실행되므로 대기 시간과 토큰 소모가 크게 줄어듭니다.

## Esc 두 번 — 이전 상태로 되돌리기

아이디어를 시도하다 흐름이 꼬이면 <KeyCombo keys={["Esc", "Esc"]} />를 눌러 깔끔한 지점으로 되돌립니다.

- 대화 내용만 되돌리기
- 코드만 되돌리기
- 둘 다 되돌리기

실험적인 시도를 부담 없이 할 수 있는 핵심 기능입니다.

## `/vim` — Vim 방식 프롬프트 편집

```bash
/vim
```

Vim 모드를 켜면 `h j k l` 이동, `ciw`, `dd` 같은 Vim 명령으로 프롬프트를 빠르게 수정할 수 있습니다. 마우스 없이 키보드만으로 작업할 수 있어 흐름이 끊기지 않습니다.

## <KeyCombo keys={["Ctrl", "S"]} /> — 프롬프트 임시 저장

입력 중인 프롬프트를 임시 저장합니다. 다른 메시지를 보낸 후에도 자동으로 복원됩니다.

메모장에 복사하지 않아도 되고, 생각 흐름이 끊긴 상태에서 다시 이어서 작성할 수 있습니다.

## <KeyCombo keys={["Ctrl", "R"]} /> — 프롬프트 검색

과거에 입력한 프롬프트를 역방향 검색합니다. 자주 사용하는 프롬프트를 다시 타이핑할 필요가 없습니다.

- <KeyCombo keys={["Enter"]} />로 바로 실행
- <KeyCombo keys={["Tab"]} />으로 불러와서 수정

## <KeyCombo keys={["Shift", "Tab"]} /> 두 번 — Plan 모드

코드를 수정하지 않고 읽기, 검색, 분석만 진행하는 모드입니다.

구현 전에 구조와 위험 요소를 먼저 정리할 수 있어, 한 번에 제대로 실행하기 좋은 준비 단계입니다.

<Callout type="tip">
Boris Cherny는 대부분의 세션을 Plan 모드에서 시작합니다. 좋은 플랜이 좋은 코드를 만듭니다.
</Callout>

## `@` 멘션으로 컨텍스트 추가

`@`를 입력하면 파일, 디렉토리, MCP 서버를 빠르게 대화에 추가할 수 있습니다.

```
@src/auth.ts 이 파일의 인증 로직을 설명해줘
@src/components/ 이 디렉토리의 컴포넌트 구조를 분석해줘
```

## 단축키 총정리

| 단축키 | 기능 |
|--------|------|
| `!명령어` | bash 즉시 실행 |
| Esc × 2 | 이전 상태로 되돌리기 |
| `/vim` | Vim 편집 모드 토글 |
| Ctrl+S | 프롬프트 임시 저장 |
| Ctrl+R | 프롬프트 역방향 검색 |
| Shift+Tab × 2 | Plan 모드 토글 |
| `@` | 파일/디렉토리 멘션 |
| Tab/Enter | 자동 제안 수락 |
