## 슬래시 커맨드란?

Claude Code에서 `/`로 시작하는 명령어는 특별한 기능을 실행합니다. 대화와는 별도로, Claude Code 자체의 동작을 제어합니다.

## 자주 쓰는 명령어

### `/clear` — 대화 초기화

현재 대화를 모두 지우고 새로 시작합니다. 토큰을 절약하거나 주제를 바꿀 때 유용합니다.

```bash
/clear
```

### `/compact` — 대화 요약

긴 대화를 요약하여 컨텍스트 윈도우를 절약합니다. 대화가 길어졌을 때 사용하면 토큰 소모를 줄일 수 있습니다.

```bash
/compact
```

<Callout type="tip">
대화가 길어졌다고 느끼면 `/compact`를 실행하세요. Claude가 핵심 맥락만 유지하면서 나머지를 요약합니다.
</Callout>

### `/stats` — 사용 통계

작업 횟수, 사용 패턴 등의 통계를 확인합니다. GitHub 잔디처럼 자신의 Claude Code 활용도를 파악할 수 있습니다.

```bash
/stats
```

### `/usage` — 토큰 사용량 확인

현재 토큰 사용량과 한도를 확인합니다.

```bash
/usage
```

추가 사용이 필요하면 `/extra-usage`로 제한을 해제할 수 있습니다.

### `/context` — 컨텍스트 확인

시스템 프롬프트, MCP 서버, 메모리 파일 등 현재 컨텍스트에 포함된 항목을 확인합니다. 불필요한 토큰 소모를 찾아 정리할 수 있습니다.

```bash
/context
```

### `/export` — 대화 기록 내보내기

현재 대화의 모든 프롬프트, 응답, 도구 호출을 마크다운 파일로 내보냅니다.

```bash
/export
```

작업 기록을 문서로 남기거나 학습 자료로 정리할 때 유용합니다.

### `/rename` — 세션 이름 변경

현재 세션에 이름을 붙여 나중에 쉽게 찾을 수 있게 합니다.

```bash
/rename api-migration
```

이후 `/resume api-migration`으로 바로 해당 세션을 불러올 수 있습니다.

## 명령어 총정리

| 명령어 | 설명 |
|--------|------|
| `/clear` | 대화 초기화 |
| `/compact` | 대화 요약으로 토큰 절약 |
| `/stats` | 사용 통계 확인 |
| `/usage` | 토큰 사용량 확인 |
| `/context` | 컨텍스트 구성 확인 |
| `/export` | 대화 기록 마크다운 내보내기 |
| `/rename` | 세션 이름 변경 |
| `/init` | CLAUDE.md 자동 생성 |
| `/plan` | 플랜 모드 토글 |
| `/permissions` | 권한 관리 |
| `/vim` | Vim 모드 토글 |
