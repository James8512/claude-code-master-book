## 공식 자료

가장 정확하고 최신 정보를 제공하는 공식 채널들입니다.

### Claude Code 공식 문서

Anthropic이 직접 관리하는 공식 문서는 항상 가장 먼저 참고해야 할 자료입니다. 설치부터 고급 설정까지 체계적으로 정리되어 있습니다.

- **Claude Code Documentation**: https://docs.anthropic.com/en/docs/claude-code
- **Anthropic Engineering Blog**: 기술적 깊이가 있는 공식 블로그로, Claude Code의 설계 철학과 내부 동작 원리를 다룹니다.
- **GitHub Repository**: Claude Code의 소스 코드와 이슈 트래커를 통해 최신 개발 현황을 확인할 수 있습니다.

<Callout type="info" title="공식 문서 활용 팁">
공식 문서는 빠르게 업데이트되므로, 한 달에 한 번은 방문하여 새로 추가된 섹션이 있는지 확인하세요.
</Callout>

## 커뮤니티 자료

사용자들의 실전 경험과 팁이 공유되는 커뮤니티 자원들입니다.

### ykdojo의 Claude Code Tips

ykdojo가 정리한 Claude Code 팁 모음으로, 이 책의 많은 내용이 여기에서 영감을 받았습니다. 실전 사례와 구체적인 프롬프트 예시가 풍부합니다.

- **GitHub**: github.com/ykdojo/claude-code-tips

### Ado의 Advent of Claude

Ado(@adocomplete)가 진행한 Advent of Claude 시리즈로, 25일간 매일 하나의 Claude Code 고급 팁을 다룹니다. 특히 시스템 프롬프트 분석, 숨겨진 기능, 고급 워크플로우에 대한 깊이 있는 콘텐츠입니다.

### r/ClaudeAI 서브레딧

Reddit의 Claude AI 커뮤니티로, 전 세계 사용자들이 질문, 팁, 워크플로우, 트러블슈팅을 공유합니다.

- **Reddit**: r/ClaudeAI

## 아티클

깊이 있는 분석과 실전 경험을 다루는 기술 아티클들입니다.

| 출처 | 주제 | 특징 |
|------|------|------|
| Jacob's Tech Tavern | Claude Code 심층 분석 | 기술적 깊이, 상세한 비교 |
| Pragmatic Engineer | AI 코딩 도구 트렌드 | 업계 관점, 생산성 분석 |
| Lenny's Newsletter | AI 제품 개발 | 제품 관점, 비즈니스 임팩트 |

<Callout type="tip" title="아티클 활용법">
기술 아티클은 특정 주제를 깊이 다루므로, 자신이 현재 학습 중인 주제와 관련된 아티클을 선택적으로 읽는 것이 효율적입니다.
</Callout>
