## Claude Code와 Git

Claude Code는 Git을 깊이 이해합니다. 커밋 메시지 작성부터 복잡한 머지 충돌 해결까지, Git 워크플로우의 거의 모든 부분을 자동화할 수 있습니다.

## 스마트 커밋

변경사항을 분석하고 의미 있는 커밋 메시지를 작성합니다.

```
변경사항을 확인하고 적절한 커밋 메시지로 커밋해줘
```

Claude는 `git diff`를 분석하여 변경의 목적을 파악하고, 컨벤셔널 커밋 형식에 맞는 메시지를 생성합니다.

<Callout type="tip">
CLAUDE.md에 커밋 메시지 컨벤션을 명시하면 더 일관된 커밋 메시지를 생성합니다.
</Callout>

## PR 생성

```
현재 브랜치의 변경사항으로 PR을 생성해줘.
제목은 한국어로, 본문에는 변경사항 요약과 테스트 방법을 포함해줘.
```

Claude는 변경 파일을 분석하고, GitHub CLI(`gh`)를 사용하여 PR을 생성합니다.

## 머지 충돌 해결

```
머지 충돌을 확인하고 해결해줘.
main 브랜치의 변경사항을 우선하되, 새로 추가된 기능은 유지해줘.
```

Claude는 충돌 마커를 분석하고, 양쪽의 의도를 파악하여 적절히 병합합니다.

## 릴리즈 노트 자동 생성

```
v1.2.0부터 현재까지의 커밋을 분석해서 릴리즈 노트를 작성해줘.
기능 추가, 버그 수정, 성능 개선으로 분류해줘.
```

## 유용한 Git 프롬프트 패턴

```bash
# 변경사항 요약
git diff | claude -p "이 변경사항을 요약해줘"

# 코드 리뷰
git diff main...feature | claude -p "이 PR의 코드를 리뷰해줘"

# 브랜치 정리
claude -p "머지된 브랜치를 정리해줘"
```
