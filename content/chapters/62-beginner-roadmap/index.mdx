## 1-3개월 학습 로드맵

Claude Code를 처음 시작하시나요? 이 로드맵은 12주 동안 단계적으로 Claude Code의 핵심 기능을 익히도록 설계되었습니다. 각 단계를 완료한 후에 다음 단계로 넘어가세요.

## 주차별 학습 계획

| 주차 | 주제 | 핵심 목표 |
|------|------|-----------|
| 1주차 | 설치 및 기본 명령어 | Claude Code 설치, 기본 대화, 파일 읽기/쓰기 |
| 2주차 | CLAUDE.md와 /init | 프로젝트 설정, CLAUDE.md 작성, /init으로 초기화 |
| 3-4주차 | 컨텍스트 관리 | 효과적인 프롬프트 작성, 컨텍스트 크기 관리 |
| 5-6주차 | Git 통합 | 커밋, PR 리뷰, diff 분석, 브랜치 관리 |
| 7-8주차 | 별칭과 단축키 | 터미널 별칭 설정, 키보드 단축키 활용 |
| 9-12주차 | 음성 코딩과 커스텀 명령어 | 음성 전사 도구 설정, 슬래시 커맨드 작성 |

## 단계별 상세 가이드

<Steps>
<Step title="1주차: 설치 및 기본 명령어">
가장 먼저 Claude Code를 설치하고 기본적인 사용법을 익힙니다.

```bash
# 설치
npm install -g @anthropic-ai/claude-code

# 첫 실행
claude

# 기본 요청
> 이 프로젝트의 구조를 설명해줘
> src/index.ts 파일을 읽고 요약해줘
```

이번 주의 목표는 Claude Code와 자연스럽게 대화하는 것에 익숙해지는 것입니다.
</Step>
<Step title="2주차: CLAUDE.md와 /init">
프로젝트에 CLAUDE.md를 설정하여 Claude Code에게 프로젝트 맥락을 제공합니다.

```bash
# /init으로 프로젝트 초기화
> /init
```

CLAUDE.md에 기술 스택, 코딩 컨벤션, 프로젝트 구조를 기록하세요.
</Step>
<Step title="3-4주차: 컨텍스트 관리">
효과적인 프롬프트를 작성하고, 컨텍스트 창을 효율적으로 활용하는 방법을 배웁니다. 너무 많은 정보를 한 번에 제공하면 오히려 성능이 떨어진다는 것을 이해하게 됩니다.
</Step>
<Step title="5-6주차: Git 통합">
Claude Code를 Git 워크플로우에 통합합니다. 커밋 메시지 생성, PR 리뷰 요청, 코드 변경사항 분석 등을 연습합니다.
</Step>
<Step title="7-8주차: 별칭과 단축키">
반복적인 명령어를 별칭으로 설정하고, 키보드 단축키를 활용하여 작업 속도를 높입니다.
</Step>
<Step title="9-12주차: 고급 기초">
음성 코딩을 시도하고, 자주 사용하는 작업을 커스텀 슬래시 커맨드로 만들어봅니다.
</Step>
</Steps>

## 성공 지표

3개월 후 다음 항목들을 달성했다면 초급 단계를 성공적으로 완료한 것입니다.

- Claude Code를 매일 사용하는 습관이 형성됨
- CLAUDE.md를 작성하고 유지할 수 있음
- 기본적인 코드 생성, 수정, 리뷰를 Claude Code로 수행할 수 있음
- Git 워크플로우에 Claude Code를 자연스럽게 통합함
- 최소 3개 이상의 터미널 별칭을 설정하여 사용 중
