## Vim 모드 활성화

Vim 사용자라면 `/vim` 명령어로 Vim 모드를 활성화할 수 있습니다. 프롬프트 입력 박스에서 Vim 키 바인딩을 사용할 수 있습니다.

## 사용 가능한 명령

| 명령 | 기능 |
|------|------|
| `hjkl` | 커서 이동 |
| `w`, `b` | 단어 단위 이동 |
| `0`, `$` | 줄 시작/끝으로 이동 |
| `dd` | 줄 삭제 |
| `yy` | 줄 복사 |
| `p` | 붙여넣기 |
| `i`, `a` | 삽입 모드 |
| `Esc` | 일반 모드 |

<Callout type="tip" title="Ado의 조언">
"Vim 사용자라면 이 기능 없이는 살 수 없을 겁니다. 긴 프롬프트를 편집할 때 특히 유용합니다."
</Callout>
